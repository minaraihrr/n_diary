# N年日記
指定した日付について、過去N年分の日記を一覧・編集できるWebアプリです。
<img src="resources/demo.gif" width="200">

## What is this
いわゆる「N年日記」をWebアプリとして実装しています。  
同じ日付の過去ログを並べて振り返れることを目的としています。

## Tech Stack
- Node.js
- Express
- PostgreSQL
- Vanilla JavaScript
